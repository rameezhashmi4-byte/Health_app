# Font Setup Instructions

## Adding Sofia Pro Font

To use the custom Sofia Pro font, you need to add the font files to the project:

1. Download Sofia Pro font files from a font provider (e.g., Adobe Fonts, Google Fonts alternative)
2. Place the following font files in `app/src/main/res/font/`:
   - `sofia_pro_regular.ttf` (or `.otf`)
   - `sofia_pro_medium.ttf`
   - `sofia_pro_semibold.ttf`
   - `sofia_pro_bold.ttf`

3. The font family is already configured in `font_family.xml`

**Alternative:** If you don't have Sofia Pro, the app will fall back to the system font (Roboto on Android). You can also use Inter or similar modern sans-serif fonts as alternatives.

## Font Licensing

Make sure you have proper licensing for Sofia Pro if using it commercially.
